# Chapter 2. Creating Your Repository
1. Create a repository then write code 
- Create a repository
- Pull the repository to your computer 

```
$git clone HTTPS/SSH
```

- Add the files 

```
$git add .
```

- Commit at the command line

```
$git commit -m "commit"
```

- Push code 

```
$git push origin branch
```

2. Create local project and push

- Create a repository
- Initialize git file

```
$git init 
```
